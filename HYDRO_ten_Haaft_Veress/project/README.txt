------------------------------------------------------------------------
0. Quellcode
------------------------------------------------------------------------
-in hydro.cpp sind alle Methoden der Simulation
------------------------------------------------------------------------
1. Linux: Kompilierung mit g++
------------------------------------------------------------------------
-Es wird g++ benötigt: 
 bash: sudo apt-get install g++
-einfach im bash bspw.  g++ hydro.cpp -o run_simulations
 und dann ./run_simulations
------------------------------------------------------------------------
2. Windows: Kompilierung und starten der Simulationen über Visual Studio
------------------------------------------------------------------------
-in Visual Studio (nicht VS Code) kann die Projektmappe "Hydrodynamics_VS_project.sln"
 geöffnet werden
-VS sollte dann die Dateien automatisch kompilieren
------------------------------------------------------------------------
3. Ergebnisse
------------------------------------------------------------------------
-Die generierten Daten befinden sich in dem "results" Ordner
